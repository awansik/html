Welcome to Python
Python is a high-level programming language for general-purpose programming. It was created by Guido van Rossum and first released in 1991.
Python's design philosophy emphasizes code readability, most notably using significant whitespace and providing constructs that enable clear programming.
In July 2018, Van Rossum stepped down as the leader in the language community after 30 years. Python features a dynamic type system and automatic memory management.
It supports multiple programming paradigms, including object-oriented, imperative, functional and procedural, and has a large and comprehensive standard library.
Python interpreters are available for many operating systems. CPython, the reference implementation of Python, is open source software and has a community-based
development model, as do nearly all of Python's other implementations. Python and CPython are managed by the non-profit Python Software Foundation.
